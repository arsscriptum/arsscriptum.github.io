Development Package Ready

A fully functional Deja Insight development package along with the license and the log management app is available for download here:

DejaInsight.SDK.zip.aes

Instructions for unpacking. The file is encrypted.

  • Download the file
  • Decrypt with AES to Zip file, using the
    Win32.AesCrypt.Console
    compiled exe.
  • AesCryptConsole.exe -d ".\DejaInsight.SDK.zip.aes"
  • After the file is decrypted, it will be in a zip format. You need to expand the archive on your development drive, using the second password.
